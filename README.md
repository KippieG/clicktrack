# ClickTrack Marketing Page

Static marketing page for ClickTrack, a private macOS menu bar app for tracking clicks, keystrokes, scrolls, focus sessions, streaks, and achievements.

## Local Preview

Open `index.html` directly in a browser. No build step is required.

## GitHub Pages

Use GitHub Pages with:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`

The live page will be available at:

`https://kippieg.github.io/clicktrack-marketing/`
