02

Setup

ClickTrack needs macOS Accessibility permission to count global clicks, keystrokes, and scrolls.

03

Privacy

ClickTrack stores data locally, never records typed content, and does not require an account.

What ClickTrack tracks

  • Clicks, keystrokes, scrolls, CPM, WPM, and active minutes
  • Focus sessions, streaks, achievements, weekly history, and heatmaps
  • Counts and app usage only, not typed text or private content

Current build status

  • The GitHub download is a developer beta and is not notarized yet.
  • macOS may require right-click > Open on first launch.
  • Pro purchases are not active in the direct-download build yet.
